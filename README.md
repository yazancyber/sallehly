# صلّحلي V23 Secure Responsive

نسخة محسّنة تركّز على الأمان والتوافق مع الهاتف:

- لا تظهر بيانات دخول الإدارة داخل الواجهة.
- لا توجد كلمة سر أدمن افتراضية داخل الكود.
- إنشاء حساب الإدارة يتم من ملف `.env` فقط.
- `.gitignore` يمنع رفع `.env` وقاعدة البيانات والملفات المرفوعة إلى GitHub.
- تصميم متجاوب للموبايل والكمبيوتر.
- لوحة العميل، الفني، والإدارة بصلاحيات منفصلة.
- شات محمي يمنع أرقام الهاتف وواتساب وتيليجرام والإيميلات وروابط التواصل الخارجي فقط.
- رفع صورة المشكلة اختياري مع معاينة.
- تحديث مباشر عبر Socket.IO مع Polling احتياطي.

## التشغيل محلياً

1. ثبّت الحزم:

```bash
npm install
```

2. انسخ ملف البيئة:

```bash
copy .env.example .env
```

3. افتح `.env` وضع بياناتك:

```env
JWT_SECRET=مفتاح_طويل_وعشوائي
ADMIN_EMAIL=بريد_الإدارة
ADMIN_PASSWORD=كلمة_سر_قوية
```

4. شغّل المشروع:

```bash
npm start
```

ثم افتح:

```text
http://localhost:3000
```
