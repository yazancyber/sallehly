# ملاحظات أمان صلّحلي

- لا توجد بيانات دخول أدمن داخل الواجهة أو الكود.
- أنشئ ملف `.env` من `.env.example` وضع `ADMIN_EMAIL` و `ADMIN_PASSWORD` و `JWT_SECRET`.
- لا ترفع `.env` أو مجلد `data/` أو `public/uploads/` إلى GitHub.
- عند أول تشغيل، يتم إنشاء حساب الإدارة فقط إذا كانت متغيرات الإدارة موجودة في `.env`.
- إذا لم يتم إنشاء أدمن، أوقف السيرفر، أضف بيانات الإدارة في `.env`، ثم شغّل السيرفر من جديد.
